<strong>이타적 자서전을 통해 세대를 연결하는 사회적 가치 창출 플랫폼</strong><br/>
봉사센터와의 연계를 통해 봉사활동을 쉽게 관리하고<br/>

## 팀 소개

| **오시은** | **강주은** | **성윤정** | **정지윤** | **이효림** |
|:-----------:|:-----------:|:-----------:|:-----------:|:-----------:|

## 사용 기술

- **Frontend**: React, HTML/CSS, JavaScript  
- **Backend**: Spring Framework  
- **Database**: mySQL  
- **Design Tools**: Figma  
- **배포**:
  - 프론트엔드: vercel
  - 백엔드: Aws Ec2(NginX, Let's encrypt를 통한 SSL 인증서)
